Accessing the Platform
    - To access AWS Cloud Services, you can use
        - AWS Management Console
        - AWS Command Line Interface (CLI)
        - AWS Software Development Kit (SDK)
    - The AWS Management Console is a web application for managing AWS Cloud services.
        - The console provides an intuitive user interface for performing tasks
        - Each service has its own console, which can be accessed from AWS Management Console.
        - This console also provides information about the account and billing
    - The AWS Command Line Interface (CLI) is a unified tool used to manage AWS Cloud Services.
        - You are able to control multiple services from the command line and automate them through scripts.
    - The AWS Software Development Kits (SDKs) provide an API that interacts with the web services that makes up the AWS Platform.
        - The SDK provide support for many different programming languages and platforms
        - You can make HTTP calls directly to the web service endpoints, but using the SDK provides the abstraction on communicating with the services.
